# awesome-mycel

## Implementations
Each implementation declares the latest Mycel version it supports.

The first number in the version (for example, the 3 in 3.0) shows the major version. All 3.x versions of Mycel are compatible with each other, with only new features or bug fixes changing between them.


| Name | Type | Supported Mycel version | Stars |
|------|------|-------------------------|--------|
| [Mycelium](#Mycelium) | Application | 0.1 | [![GitHub stars](https://img.shields.io/github/stars/mycel-project/mycelium?style=social)](https://github.com/mycel-project/mycelium)|

### [Mycelium](https://github.com/mycel-project/mycelium)

Mycelium is a cross-platform application (Windows, Linux, Android. Apple support coming soon) that provides a seamless implementation of Mycel.

It focuses on delivering a smooth experience with minimal friction and consistent behavior across supported platforms.
